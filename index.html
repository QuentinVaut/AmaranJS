Basic Usage

Basic usage without any theme setted.It will use default theme

    
$.amaran({
    content:{
        title:'Your Download is Ready!',
        message:'1.4 GB',
        info:'my_birthday.mp4',
        icon:'fa fa-download'
    },
    theme:'awesome ok'
 });

Using Predefined Themes

AmaranJS has 4 predefined themes each of them has easy unique setup.Lets see with Awesome theme.

    
$.amaran({
    content:{
        title:'Your Download is Ready!',
        message:'1.4 GB',
        info:'my_birthday.mp4',
        icon:'fa fa-download'
    },
    theme:'awesome ok',
    position:'top left'
});

Setting Position

Just set position option with vertical and horizontal position order.Like top left or bottom right

    
$.amaran({
    content:{
        title:'Your Download is Ready!',
        message:'1.4 GB',
        info:'my_birthday.mp4',
        icon:'fa fa-download'
    },
    theme:'awesome ok',
    position:'top left',
    inEffect:'slideRight',
    outEffect:'slideLeft'
});

Setting Effects

We have 5 built-in effects ( fade, slideLeft , slideRight, slideTop, slideBottom ) and two options to use animate.css effects.

For appering effect you can use inEffect option and disappering effect you can use inEffect option.

If you want use animate.css effects we have cssanimationIn and cssanimationOut options.

Installation

You can install AmaranJS through bower:

    
bower install amaranjs

Or download AmaranJS and go to dist directory and copy all the folders to a new destination, which will be your project home.

It’s not required, but i recommend placing CSS files in and JavaScript files and initialization code in the footer of your site (before the closing </body> tag).

jQuery required for Amaran JS.If you already have jquery.js on your site, don’t include it a second time, or use jQuery.noConflict(); mode

    
<link rel="stylesheet" href="dist/css/amaran.min.css">

You can add jQuery via Google CDN

    
<script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>

Place Amaran JS after jQuery

    
<script src="dist/js/jquery.amaran.js"></script>

So minimalistic setup look like this

    
<!DOCTYPE html>
<html>
    <head>
        <title>My Web Page</title>
        <link rel="stylesheet" href="dist/css/amaran.min.css">
    </head>
    <body>
        <p>My Content</p>
        <script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
        <script src="dist/js/jquery.amaran.js"></script>
    </body>
</html>

Optional Steps

Awesome theme uses fontawesome if you want to use awesome theme you have to include fontawesome.

    
<link href="//netdna.bootstrapcdn.com/font-awesome/4.0.3/css/font-awesome.css" rel="stylesheet">

If you want to use Animate.css effects you have to include Animate.css . Animate.css already included dist/css folder.

    
<link rel="stylesheet" href="dist/css/animate.min.css">
